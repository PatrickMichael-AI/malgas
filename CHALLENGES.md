# The Oak and Barrel — UI/UX Challenge

Welcome! This is an intentionally bland restaurant website. Your mission: **make it beautiful**.

Use whatever tools you want — AI coding agents, manual coding, CSS frameworks, component libraries — the only rule is the final result must still be a functional React site.

---

## Beginner Challenges

- [ ] Change the font from Times New Roman to something modern (Google Fonts)
- [ ] Pick a real color scheme (not gray and white)
- [ ] Add hover effects to the navigation links
- [ ] Fix the broken image on the home page (hint: check the file names)
- [ ] Make the contact form actually look like a form (labels, spacing, alignment)
- [ ] Add a favicon
- [ ] Style the menu items as cards instead of plain text
- [ ] Remove the `<hr>` tags and use better visual section dividers

## Intermediate Challenges

- [ ] Make the site responsive (mobile-friendly)
- [ ] Add a 404 "Page Not Found" page
- [ ] Create a proper hero section with text overlay on the home page image
- [ ] Add a sticky/fixed navigation bar
- [ ] Style the testimonials section with star ratings and card layouts
- [ ] Add smooth scrolling and page transitions
- [ ] Create a proper image gallery with consistent sizing
- [ ] Add a "Back to Top" button
- [ ] Fix the CSS specificity conflicts between pages

## Advanced Challenges

- [ ] Extract the hardcoded menu data into a separate data file and render with `.map()`
- [ ] Add menu filtering/search (by category, price range, dietary info)
- [ ] Implement dark mode with a toggle
- [ ] Add form validation to the contact/reservation form
- [ ] Add Open Graph meta tags for social media sharing
- [ ] Implement lazy loading for images
- [ ] Add animations and micro-interactions (subtle, tasteful ones)
- [ ] Run a Lighthouse audit and fix all accessibility issues
- [ ] Make the navigation highlight the active page

## Expert Challenges

- [ ] Add a reservation system using localStorage
- [ ] Implement a shopping cart / order system
- [ ] Add Framer Motion or similar for page transitions
- [ ] Create a theme switcher with multiple color schemes
- [ ] Add internationalization (multi-language support)
- [ ] Achieve a perfect Lighthouse score (100/100/100/100)
- [ ] Deploy to Vercel/Netlify with a custom domain

---

Good luck! Share your results with the community.
